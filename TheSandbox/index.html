Site Information

Welcome to the Sandbox! This is my site for CSC 372. It will contain various browser games, information, tips, projects, and tools geared towards expanding interest in computer science. Its purpose will be to generate interest in computer programming, especially for beginners, but will hopefully be engaging for more advanced levels of programmers as well. My primary audience will be anyone who wants to learn about programming. There will also be an emphasis on being accessible to a younger audience, maybe middle school and up. My site will include dynamic content through interactive JavaScript and CSS. There will be animated buttons, animated game windows, mobile compatibility, and much more. Hopefully, by showing how you can make a page interactive, my audience will be interested in learning more about it.
