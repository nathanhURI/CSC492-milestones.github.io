Stroke refers to the lines themselves. Fill refers to the color inside certain shapes (like circles and squares). Background is the background color. Make sure to keep transparency values between 0 and 255. You can also resize the canvas to whatever you want, but your design won't look the same after changing to a different size because of how things rotate. Also try to keep the shape count low, the program gets laggy when it gets large. The pencil and marker tool are a work in progress, so don't go nuts with em.
